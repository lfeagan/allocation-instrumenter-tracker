The Allocation Instrumenter Tracker is a small library to simplify tracking memory allocations during method execution with [allocation-instrumenter](https://github.com/google/allocation-instrumenter). It consists of a single class, `MemoryAllocationTracker`, that tracks method execution time and the memory allocated during execution.

## Dependencies
1. [allocation-instrumenter](https://github.com/google/allocation-instrumenter)
1. [AspectJ](https://github.com/eclipse-aspectj/aspectj)
2. [Wheat](https://github.com/lfeagan/wheat)
3. [SLF4J](https://www.slf4j.org/)

## Test Dependencies
